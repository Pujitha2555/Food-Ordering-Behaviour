# Food Ordering Behaviour and Consumer Trends – A Structured Analysis of Choices and Habits

## Project Overview
This project analyzes customer food ordering behaviour and consumer trends using Tableau. It provides insights into customer preferences, popular cuisines, meal choices, city-wise ordering patterns, and factors influencing food ordering decisions through interactive dashboards and stories.

## Objectives
- Analyze customer food ordering behaviour.
- Identify popular cuisines and meal preferences.
- Study city-wise ordering trends.
- Create interactive Tableau dashboards and stories.
- Generate business insights from the dataset.
  
## Features
- Interactive Tableau Dashboard
- Order Value Analysis
- Cuisine Preference Analysis
- City-wise Order Distribution
- Meal Type Analysis
- Delivery Time Analysis
- Peak Ordering Time Analysis
- Business Performance Insights
